# projet_webavance
Contrôle Continu - TP web (noté)


Vous allez réaliser un mini site destiné à gérer un panier d’articles en hifi et électroménager.

L’application attendue est réalisée avec les technologies PHP, HTML / CSS, et Javascript.

Vous fournirez comme livrables :

    un fichier compressé en zip ou 7z contenant votre code source qui pourra être installé sur un serveur web (dans le répertoire /www )

    un rapport simple (max 5-10 pages) où vous expliquerez :
      - les choix que vous avez effectués pour mettre en œuvre le site
      - les points sur lesquels vous avez rencontrés des difficultés et comment vous les avez résolus – ou à défaut si des points restent non résolus : quelle analyse pouvez-vous en faire si vous deviez les résoudre par la suite.


Si vous avez aussi une url pour tester l’application vous pouvez la fournir(ce n’est pas obligatoire)
